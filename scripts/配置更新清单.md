## 📋 需要更新的文件清单

### 1. Vue前端 (vue-front/)

#### package.json

```diff
{
-  "name": "kaoyan-platform-front",
+  "name": "kaoyan-frontend",
   "version": "1.0.0",
   ...
}
```

#### vite.config.js

如果有引用后端API的代理配置，检查是否需要更新：

```js
export default defineConfig({
  server: {
    proxy: {
      '/api': {
-       target: 'http://localhost:8080',  // 如果地址不变则保持
+       // 保持不变
        changeOrigin: true
      }
    }
  }
})
```

#### src/router/index.js

检查路由配置，如果涉及路径引用：

```js
const routes = [
  {
    path: '/',
    // 保持不变
  }
]
```

#### .env / .env.production

如果有环境变量配置：

```env
# 保持不变
VITE_API_BASE_URL=http://localhost:8080/api
```

## 🔍 自动化更新脚本

### PowerShell脚本 (Windows)

```powershell
# scripts/update-configs.ps1

Write-Host "开始更新配置文件..." -ForegroundColor Green

# 1. 更新 package.json
Write-Host "[1/5] 更新 vue-front/package.json..." -ForegroundColor Yellow
$json = Get-Content "vue-front/package.json" -Raw | ConvertFrom-Json
$json.name = "kaoyan-frontend"
$json | ConvertTo-Json -Depth 10 | Set-Content "vue-front/package.json"

# 2. 更新 pom.xml
Write-Host "[2/5] 更新 java-back/pom.xml..." -ForegroundColor Yellow
(Get-Content "java-back/pom.xml") -replace 'KaoYanPlatform-back', 'java-back' | Set-Content "java-back/pom.xml"

# 3. 更新 Python相关
Write-Host "[3/5] Python配置保持不变" -ForegroundColor Green

# 4. 更新 README
Write-Host "[4/5] 更新 README.md..." -ForegroundColor Yellow
# 手动更新文档

Write-Host "[5/5] 配置更新完成！" -ForegroundColor Green
```

### Bash脚本 (Linux/Mac)

```bash
#!/bin/bash
# scripts/update-configs.sh

echo "开始更新配置文件..."

# 1. 更新 package.json
echo "[1/5] 更新 vue-front/package.json..."
sed -i 's/"name": "kaoyan-platform-front"/"name": "kaoyan-frontend"/' vue-front/package.json

# 2. 更新 pom.xml
echo "[2/5] 更新 java-back/pom.xml..."
sed -i 's/KaoYanPlatform-back/java-back/g' java-back/pom.xml

# 3. 更新 application.yml
echo "[3/5] 更新 java-back配置..."
find java-back -name "application*.yml" -o -name "application*.yaml" | while read file; do
    sed -i 's/KaoYanPlatform/kaoyan-platform/g' "$file"
done

echo "配置更新完成！"
```
