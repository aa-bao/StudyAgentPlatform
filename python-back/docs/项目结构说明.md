## 📁 核心目录详解

### 1. app/api/v1/ - API路由层

**职责：**
- 定义HTTP接口
- 验证请求参数
- 调用服务层处理业务
- 返回标准化响应

**questions.py 示例：**
```python
from fastapi import APIRouter, HTTPException
from app.models.schemas import ExtractRequest, ExtractResponse
from app.services.extract_service import ExtractService

router = APIRouter()
extract_service = ExtractService()

@router.post("/extract", response_model=ExtractResponse)
async def extract_questions(request: ExtractRequest):
    """
    提取题目接口

    流程：
    1. 验证请求参数（Pydantic自动完成）
    2. 调用ExtractService处理业务
    3. 返回标准化响应
    """
    try:
        result = await extract_service.extract_questions(
            request.content,
            request.use_glm
        )
        return ExtractResponse(
            success=True,
            message="提取成功",
            data=result
        )
    except Exception as e:
        raise HTTPException(status_code=500, detail=str(e))
```

### 3. app/services/ - 业务逻辑层

**职责：**
- 实现核心业务逻辑
- 编排多个服务调用
- 处理异常和数据转换

**extract_service.py 示例：**
```python
from app.models.schemas import QuestionImportDTO
from app.services.glm_service import GLMService
from app.services.prompt_service import PromptService
import logging

logger = logging.getLogger(__name__)

class ExtractService:
    """题目提取服务"""

    def __init__(self):
        self.glm_service = GLMService()
        self.prompt_service = PromptService()

    async def extract_questions(
        self,
        content: str,
        use_glm: bool = True
    ) -> QuestionImportDTO:
        """
        提取题目（主入口）

        业务流程：
        1. 验证内容长度
        2. 调用GLM服务提取
        3. 解析和验证结果
        4. 返回标准化数据
        """
        # 1. 验证
        if len(content) > settings.MAX_CONTENT_LENGTH:
            raise ContentTooLongException(settings.MAX_CONTENT_LENGTH)

        # 2. 调用GLM
        logger.info(f"开始提取，use_glm={use_glm}")
        json_response = await self.glm_service.extract(content, use_glm)

        # 3. 解析
        questions = self._parse_response(json_response)

        # 4. 返回
        return QuestionImportDTO(questions=questions)
```

```
  - 接收HTTP请求

  ↓
业务逻辑层 (app/services/extract_service.py)

  - 调用GLM服务
  ↓

  - 调用智谱API
  - 返回原始响应
  ↓

  ↓

  - 构建响应
  ↓

```

## 📝 关键文件说明

### main.py（项目根目录）

```python
"""
应用启动入口

职责：
1. 创建FastAPI应用实例
2. 配置中间件（CORS等）
3. 注册路由
4. 配置异常处理器
"""

from fastapi import FastAPI
from fastapi.middleware.cors import CORSMiddleware
from app.api.v1 import questions, health
from app.core.exceptions import AIServiceException
from fastapi.responses import JSONResponse

# 创建应用
app = FastAPI(
    title="考研平台AI服务",
    description="提供题目提取、模型训练等AI功能",
    version="1.0.0"
)

# 配置CORS
app.add_middleware(
    CORSMiddleware,
    allow_origins=["*"],
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)

# 注册路由
app.include_router(questions.router, prefix="/api/v1")
app.include_router(health.router)

# 异常处理器
@app.exception_handler(AIServiceException)
async def ai_exception_handler(request, exc):
    return JSONResponse(
        status_code=400,
        content={"success": False, "message": exc.message}
    )

@app.get("/")
def read_root():
    return {
        "message": "考研平台AI服务",
        "docs": "/docs",
        "version": "1.0.0"
    }

# 启动命令：uvicorn main:app --reload
```
