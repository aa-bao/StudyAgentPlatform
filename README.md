# 项目介绍
这项目是我考完研做的毕设，所以原定的主题为考研学习平台，但其实这个系统我搭建的是一个框架，只需要自己收集好数据替换，它就能成为任意的学习平台，如考公、考编、亦或者是什么证书考试。

后续有时间还会完善数据部分。

智能批改部分只专门针对考研的，里面的内置prompt也对考研范围的适用性更强。

# 特点
实现 LLM 智能批改。

实现智能推题，llm 推的也可能不智能...

实现 LLM 答疑，我导入了经典的教材和大纲，用RAG搭建答疑专家Agent，大幅降低幻觉率。
