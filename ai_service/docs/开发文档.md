## 1. 项目概述

### 1.1 项目背景

考研平台AI服务是一个独立的Python微服务，负责处理所有AI相关功能，包括：
- 题目智能提取（使用GLM大模型）
- 模型训练（未来功能）
- 数据处理（未来功能）
- 推荐系统（未来功能）

### 1.2 设计原则

1. **职责单一**：只负责AI和数据业务逻辑，文件管理、权限控制等由Java后端处理
2. **高可用性**：GLM调用失败时，直接返回错误提示，不降级处理
3. **易扩展性**：模块化设计，便于添加新的AI功能
4. **配置灵活**：Prompt模板等配置化管理，无需修改代码

### 1.3 系统定位

```
┌─────────────────────────────────────────────┐
│            Java主系统                        │
│  - 用户管理、权限控制                        │
│  - 题库管理、试卷管理                        │
│  - 文件上传、图片处理                        │
│  - 业务流程编排                              │
└──────────────┬──────────────────────────────┘
               │ HTTP REST API
               ↓
┌─────────────────────────────────────────────┐
│         Python AI服务 (本服务)              │
│  - 题目智能提取 (GLM)                       │
│  - 模型训练                                  │

└─────────────────────────────────────────────┘
```

### 4.1 题目提取模块（优先级：高）

- 使用智谱GLM-4-Flash模型进行智能提取
- Prompt模板化管理

- `POST /api/v1/questions/preview` - 预览提取结果

**输入：**
```json
{
  "content": "Markdown格式的题目内容",
  "use_glm": true
}
```

**输出：**
```json
{
  "questions": [
    {
      "questionNumber": 1,
      "type": 1,
      "content": "题干内容",
      "options": ["A. 选项1", "B. 选项2"],
      "answer": "A",
      "analysis": "解析内容",
      "tags": ["数据结构", "栈"]
    }
  ]
}
```

## 6. 数据模型

### 6.1 Pydantic模型定义

```python
# app/models/schemas.py

from pydantic import BaseModel, Field
from typing import List, Optional

class QuestionItem(BaseModel):
    """单个题目模型"""

    questionNumber: int = Field(
        ...,
        description="题号，用于试卷导入时的排序",
        ge=1
    )

    type: int = Field(
        ...,
        description="题目类型：1=单选, 2=多选, 3=填空, 4=简答",
        ge=1,
        le=4
    )

    content: str = Field(
        ...,
        description="题干内容，支持Markdown格式",
        min_length=1
    )

    options: List[str] = Field(
        default_factory=list,
        description="选项列表，选择题必填"
    )

    answer: str = Field(
        ...,
        description="答案：单选填字母，多选用逗号分隔"
    )

    analysis: str = Field(
        default="",
        description="解析内容"
    )

    tags: List[str] = Field(
        default_factory=list,
        description="标签列表，如['数据结构', '栈']"
    )

    class Config:
        json_schema_extra = {
            "example": {
                "questionNumber": 1,
                "type": 1,
                "content": "为解决计算机主机与打印机之间速度不匹配问题...",
                "options": [
                    "A. 栈",
                    "B. 队列",
                    "C. 树",
                    "D. 图"
                ],
                "answer": "B",
                "analysis": "打印数据缓冲区需要按照先进先出的原则...",
                "tags": ["数据结构", "队列"]
            }
        }

class QuestionImportDTO(BaseModel):
    """题目导入DTO"""
    questions: List[QuestionItem]

    class Config:
        json_schema_extra = {
            "example": {
                "questions": [
                    {
                        "questionNumber": 1,
                        "type": 1,
                        "content": "题干内容",
                        "options": ["A. 选项1"],
                        "answer": "A",
                        "analysis": "解析",
                        "tags": ["标签"]
                    }
                ]
            }
        }

class ExtractRequest(BaseModel):
    """题目提取请求"""
    content: str = Field(..., description="Markdown内容")
    use_glm: bool = Field(default=True, description="是否使用GLM")

class ExtractResponse(BaseModel):
    """题目提取响应"""
    success: bool
    message: str
    data: Optional[QuestionImportDTO] = None
```

### 6.2 字段验证规则

| 字段 | 验证规则 | 说明 |
|------|----------|------|
| questionNumber | >= 1 | 题号必须大于0 |
| type | 1-4 | 只能是1、2、3、4 |
| content | 非空 | 题干不能为空 |
| options | 列表 | 必须是字符串数组 |
| answer | 非空 | 答案不能为空 |

## 8. 错误处理

### 8.1 异常定义

```python
# app/core/exceptions.py

class AIServiceException(Exception):
    """AI服务基础异常"""
    def __init__(self, message: str, code: str = "UNKNOWN_ERROR"):
        self.message = message
        self.code = code
        super().__init__(self.message)

class GLMException(AIServiceException):
    """GLM调用异常"""
    def __init__(self, message: str):
        super().__init__(message, code="GLM_API_ERROR")

class ContentTooLongException(AIServiceException):
    """内容过长异常"""
    def __init__(self, max_length: int):
        message = f"内容超过最大长度限制：{max_length} 字符"
        super().__init__(message, code="CONTENT_TOO_LONG")

class ParseException(AIServiceException):
    """解析异常"""
    def __init__(self, message: str):
        super().__init__(message, code="PARSE_ERROR")

class ValidationException(AIServiceException):
    """验证异常"""
    def __init__(self, message: str):
        super().__init__(message, code="VALIDATION_ERROR")
```

### 8.2 异常处理器

```python
# app/main.py

from fastapi import FastAPI, Request
from fastapi.responses import JSONResponse
from app.core.exceptions import AIServiceException

app = FastAPI()

@app.exception_handler(AIServiceException)
async def ai_service_exception_handler(
    request: Request,
    exc: AIServiceException
):
    """处理自定义异常"""
    return JSONResponse(
        status_code=400,
        content={
            "success": False,
            "message": exc.message,
            "error_code": exc.code,
            "detail": str(exc)
        }
    )

@app.exception_handler(Exception)
async def general_exception_handler(
    request: Request,
    exc: Exception
):
    """处理未捕获的异常"""
    logger.error(f"未处理的异常: {exc}", exc_info=True)
    return JSONResponse(
        status_code=500,
        content={
            "success": False,
            "message": "服务器内部错误",
            "error_code": "INTERNAL_ERROR",
            "detail": str(exc) if settings.DEBUG else "请联系管理员"
        }
    )
```

### 8.3 错误处理策略

| 错误类型 | 处理方式 | 返回给用户 |
|---------|---------|-----------|
| GLM API调用失败 | 直接返回错误 | "GLM调用失败，请联系管理员" |

## 9. 日志监控

### 9.1 日志配置

```python
# app/utils/logger.py

import logging
import sys
from pathlib import Path
from logging.handlers import RotatingFileHandler

def setup_logger(
    name: str = "app",
    level: str = "INFO",
    log_file: str = "logs/app.log"
) -> logging.Logger:
    """配置日志"""

    logger = logging.getLogger(name)
    logger.setLevel(getattr(logging, level.upper()))

    # 日志格式
    formatter = logging.Formatter(
        '%(asctime)s - %(name)s - %(levelname)s - %(message)s',
        datefmt='%Y-%m-%d %H:%M:%S'
    )

    # 控制台输出
    console_handler = logging.StreamHandler(sys.stdout)
    console_handler.setFormatter(formatter)
    logger.addHandler(console_handler)

    # 文件输出
    log_path = Path(log_file)
    log_path.parent.mkdir(parents=True, exist_ok=True)

    file_handler = RotatingFileHandler(
        log_file,
        maxBytes=10*1024*1024,  # 10MB
        backupCount=5
    )
    file_handler.setFormatter(formatter)
    logger.addHandler(file_handler)

    return logger

# 全局logger
logger = setup_logger()
```
